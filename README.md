# CodeAlpha Web Scraping Project

## Overview
Web scraping project built during CodeAlpha Data Analytics Internship.

## Libraries Used
- requests
- BeautifulSoup (bs4)
- pandas

## What it does
Scrapes quotes and authors from quotes.toscrape.com
and saves data to quotes.csv

## Files
- `scraper.py` — main scraping script
- `quotes.csv` — scraped dataset output
